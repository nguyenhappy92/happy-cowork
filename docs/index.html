Personal Cursor automation hub

Cowork with your AI, on every machine.

A single repo of skills, rules, hooks, and helper scripts for Cursor. Clone once, run ./install.sh, and every Cursor session on your machine inherits the same muscle memory.

View on GitHub Quick start 
$ git clone git@github.com:nguyenhappy92/happy-cowork.git ~/code/happy-cowork
$ cd ~/code/happy-cowork && ./install.sh

What's inside

Eight folders, one job each.

skills/

One folder per Cursor skill, each with a SKILL.md the agent auto-discovers.

rules/

Persistent coding conventions, commit style, per-language guidance.

hooks/

Event-driven automation — desktop notifications, guardrails on destructive git.

agents/

Prompt and persona definitions for specialized subagents (reviewer, planner, explorer).

mcp/

Template MCP server configs for GitHub, Linear, filesystem and more.

scripts/

Plain shell helpers invoked from the terminal: new branches, branch cleanup.

templates/

Boilerplate for PR bodies, RFCs, issue templates.

docs/

Onboarding notes, workflow recipes, and — yes — this site.

How it works

  1. 1. Clone

    git clone git@github.com:nguyenhappy92/happy-cowork.git ~/code/happy-cowork

  2. 2. Install (symlinks, no copying)

    cd ~/code/happy-cowork && ./install.sh

    skills/, rules/, and hooks/ are symlinked into ~/.cursor/. Edits in the repo take effect immediately — no reinstall needed.

  3. 3. Use

    Open Cursor and type a trigger phrase like /create-pr or "draft release notes for v1.2". The agent picks the matching skill by its description frontmatter.

Design principles

  1. Symlink, don't copy. Edits in the repo take effect immediately in ~/.cursor/.
  2. One folder per skill. Keeps diffs and discovery clean.
  3. Descriptions matter. The agent selects skills by their description frontmatter — write it like a trigger phrase.
  4. No secrets in git. Use .env.example + direnv / 1Password CLI for anything sensitive.
  5. Promote twice-repeated prompts into skills. If I type the same instructions twice, it belongs here.

Roadmap

  • justfile shortcuts (just new-skill foo).
  • Canvas dashboards for recurring analyses.
  • Shared MCP config for team members.
